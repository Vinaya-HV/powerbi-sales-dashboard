# Sales Performance Dashboard – Power BI Project

## Project Overview

This project is a multi-page Power BI dashboard designed to analyze sales performance across products, regions, and dates. It provides key business insights using interactive visuals and KPIs.

## Dashboard Pages

### Page 1 – Sales Overview

* Total Sales
* Total Profit
* Profit Margin
* Sales Trend
* Top N Products

### Page 2 – Product Performance

* Total Quantity Sold
* Average Sales per Product
* Most and Least Selling Products
* Top 5 Products

### Page 3 – Regional Sales and Order Analysis

* Sales by Region
* Profit by Region
* Order Count
* Filters for Region, Date, and Product
